Ticket API

介绍

Ticket 模块覆盖了通用的票务领域，目前可实现收费类型、免费类型、质押类型的票务资产的发行、售卖、转移、验票等。整个交易过程没有平台、三方的参与，使用智能合约完成去中心化的可信交易。我们站在通用票务领域的角度，设计了符合票务标准的数据结构，同时也为用户提供自定义属性设置的接口，方便应用扩展。
