Message API

介绍

Message模块主要包装了Carrier消息功能，我们在开发过程中预置了Carrier离线接口，待Carrier离线消息SDK提供后，进行补充，其中借鉴了Hyper的消息封装代码。（message模块api目前只支持Android）
