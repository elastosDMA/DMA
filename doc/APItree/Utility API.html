﻿# Utility API

介绍

Utility API为开发者提供一些便捷工具类，方法开发者进行快速开发

目录

  • HttpUtils (网络请求工具包)
    • 1. get请求(不带参数)
    • 2. get请求(带参数)
    • 3. post请求(不带参数)
    • 4. post请求(带参数)
    • 5. post请求(json形式)
  • FileUtils (文件管理工具包)
    • 1. 文件上传
    • 2. 获取指定目录下所有文件
    • 3. 取指定目录下所有文件夹
